# Payments

Process one-time and recurring payments.

## Create a charge

```js
const charge = await client.charges.create({
  amount: 2000,        // in cents
  currency: "usd",
  source: "tok_visa",  // test token
  description: "Order #1234",
});
```

## Capture later

```js
// Authorize only
const auth = await client.charges.create({
  amount: 5000,
  currency: "usd",
  source: "tok_visa",
  capture: false,
});

// Capture when ready
await client.charges.capture(auth.id);
```

## Refunds

```js
const refund = await client.refunds.create({
  charge: charge.id,
  amount: 1000,  // partial refund
});
```
