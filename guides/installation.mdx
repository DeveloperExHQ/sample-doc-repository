# Installation

Install the DevHQ SDK in your project.

## Requirements

- Node.js >= 18 or Python >= 3.9 or Go >= 1.21
- npm, pip, or go modules

## Node.js

```bash
npm install @devhq/sdk
```

## Python

```bash
pip install devhq
```

## Go

```bash
go get github.com/devhq/devhq-go
```

## Verify

```js
import { DevHQ } from "@devhq/sdk";
console.log(DevHQ.VERSION); // "1.0.0"
```
