# Quickstart

Get started with the API in under 5 minutes.

## Install

```bash
npm install @devhq/sdk
```

## Authenticate

```js
import { DevHQ } from "@devhq/sdk";
const client = new DevHQ({ apiKey: "YOUR_API_KEY" });
```

## Make your first call

```js
const charge = await client.charges.create({
  amount: 2000,
  currency: "usd",
  source: "tok_visa",
  description: "My first charge",
});
console.log(charge.id);
```

> **Tip:** Use test-mode keys (`sk_test_*`) while exploring — no real money moves.
